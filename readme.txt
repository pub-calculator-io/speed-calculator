The speed calculator helps find speed, distance, time using the formulas d=st, s=d/t, t=d/s. Easy to use and intuitive speed calculator.

[https://www.calculator.io/speed-calculator/](https://www.calculator.io/speed-calculator/)

== Usage ==

Add the Speed Calculator shortcode to your page, post or sidebar:

`[ci_speed_calculator]`

Add the following code to your template where you would like the Speed Calculator to appear:

`<?php display_ci_speed_calculator(); ?>`

== Installation ==

1. Upload the Speed Calculator /ci_speed_calculator/ folder to the /wp-content/plugins/ directory.
2. Activate the Speed Calculator plugin through the "Plugins" menu in WordPress.
